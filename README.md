## AskNova — Perplexity AI Clone

AskNova is a full-stack AI-powered search assistant inspired by Perplexity AI.  
It delivers real-time intelligent answers using multiple AI models, with voice input and secure authentication.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Features

## AI-Powered Answers
- Integrated with Mistral AI (fast reasoning)
- Integrated with Google Gemini (context-aware responses)

## Voice Input
- Speech-to-text support
- Real-time transcription

## Authentication System
- User signup & login
- JWT-based authentication
- Secure session handling

## Email Verification
- OTP / verification link system
- Prevents fake registrations
- Email-based account activation

## Clean UI/UX
- Inspired by Perplexity AI
- Chat-style interface
- Smooth and minimal design
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
## Tech Stack

## Frontend
React.js, Next.js, Tailwind CSS, Axios  

## Backend 
Node.js, Express.js  

## Database
MongoDB, Mongoose  

## AI Integration
Mistral AI API, Google Gemini API  

## Authentication
JWT, Email Verification (Nodemailer)  

## Voice Input 
Web Speech API (webkitSpeechRecognition) 

## Deployment
Vercel (Frontend), Render (Backend)  

## Tools
Git, GitHub, Postman, dotenv
